# Описание задачи
Проанализируйте эту фотографию меню ресторана и извлеките все пункты меню. Выведите результаты в формате JSON.

Инструкции:
1. Внимательно изучите все изображение меню целиком
2. Извлеките ВСЕ видимые элементы меню, даже если они видны частично
3. Переведите названия элементов на {userLang}
4. Распределите продукты по категориям (закуска, основное блюдо, десерт, напиток, гарнир и т.д.)
5. Извлеките цены точно так, как указано (включая символы валют)
6. Создайте краткие описания на основе видимой информации для всех блюд, кроме напитков
7. Перечислите все видимые ингредиенты
8. По возможности укажите информацию об аллергенах

Формат вывода:
[
   {
    "originalName": "точное название, указанное в меню",
    "translatedName": "переведено на {userLang}",
    "image": "placeholder",
    "category": "закуска|основное блюдо|десерт|напиток|гарнир|суп|салат|друг",
    "price": "точная цена с указанием валюты",
    "shortDescription": "описание из 2-3 предложений, основанное на информации из меню и названия блюда",
    "ingredients": ["list", "all", "visible", "ingredients"],
    "containsGluten": "yes|no|unknown",
    "containsMilk": "yes|no|unknown"
  }
]

Важные рекомендации:
- Укажите ВСЕ пункты, даже если информация неполная
- Используйте "неизвестно" для указания недостающей информации об аллергене
- Сохраняйте оригинальное оформление цен (например,, "$3.99", "€12.50")
- Если ингредиенты не указаны в списке, используйте "неизвестно" или сделайте разумные выводы из названия блюда
- Для описания используйте видимые описания меню или создавайте краткие, точные описания
- Сохраняйте точную структуру JSON для каждого товара
- Если для одного товара существует несколько размеров /цен, создайте отдельные записи
