# Описание задачи
Проанализируйте эту фотографию меню ресторана и извлеките все пункты меню.

Инструкции:
1. Внимательно изучите все изображение меню целиком
2. Переведите названия элементов на {userLang}
3. Распределите продукты по категориям (закуска, основное блюдо, десерт, напиток, гарнир и т.д.)
4. Извлеките цены точно так, как указано (включая символы валют)
5. Создайте краткие описания на основе названия для всех блюд, кроме напитков
6. Перечислите все видимые ингредиенты
7. По возможности укажите информацию об аллергенах

Формат вывода:
[
   {
    "originalName": "точное название, указанное в меню",
    "translatedName": "переведено на {userLang}",
    "image": "placeholder",
    "category": "закуска|основное блюдо|десерт|напиток|гарнир|суп|салат|друг",
    "price": "точная цена с указанием валюты",
    "shortDescription": "описание из 2-3 предложений, основанное на информации из меню и названия блюда",
    "ingredients": ["list", "all", "visible", "ingredients"],
    "containsGluten": "yes|no|unknown",
    "containsMilk": "yes|no|unknown"
  }
]

Важные рекомендации:
- Сохраняйте оригинальное оформление цен (например,, "$3.99", "€12.50")
- Если ингредиенты не указаны в списке, используйте "неизвестно" или сделайте разумные выводы из названия блюда
- Для описания используйте видимые описания меню или создавайте краткие, точные описания
- Если для одного товара существует несколько размеров /цен, создайте отдельные записи
